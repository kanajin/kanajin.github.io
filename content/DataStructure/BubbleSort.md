# 冒泡排序  
逐步比较相邻的两个数，如果前一个比后一个大，就对调他们的位置。  
## 内循环：
遍历一次数组，比较a[n]与a[n+1]的大小，根据情况执行对调，然后n+1，重复  
## 外循环：
需要遍历几次数组（执行几次内循环）才可以完成冒泡排序。值为数组长度（上界）  
## 代码：
```C
void sort(int *a, int size){
    for(int j = 0; j < size -1; j++){
        for(int i = 0; i < size - 1; i++){
            if(a[i] > a[i + 1]){
                swap(a + i, a + i + 1);
            }
        }
    }
}
```

# 插入排序  
从左到右开始，先将第一个数标记为已排序，然后将他的下一个数取出，逐个与前面的数字比较。如果取出的数字更小，则继续往左比较，直到遇到一个比他小的数字，将取出的数插入到这个数的右边  
## 代码：  
```C
void insert(int *a, int size){
    for(int i = 1; i < size; i++){
        int temp = a[i];
        int j = i - 1;
        for(; j >= 0 && a[j] > temp; j--){
            a[j + 1] = a[j]; //是将这个更大的数字右移一格
        }
        a[j + 1] = temp;
    }
}
```

**注意** 程序中的 a[j + 1] = a[j] 的意思是将更大的数字右移一格
